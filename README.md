# @banrendi/stripe-connect-wrapper

A React Native wrapper over StripeiOS and StripeAndroid to provide embedded functionality for stripe-connect

## Installation


```sh
npm install @banrendi/stripe-connect-wrapper
```


## Usage


```js
import { multiply } from '@banrendi/stripe-connect-wrapper';

// ...

const result = multiply(3, 7);
```
